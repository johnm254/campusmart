═══════════════════════════════════════════════════════════════

Problem: "password authentication failed"
Solution: Database credentials are now correct in .env

Problem: "Cannot find module 'mysql2'"
Solution: Run in backend folder: npm install mysql2

Problem: Backend won't start
Solution: Check if port 5000 is available on HostPinnacle

Problem: Frontend can't connect to backend
Solution: Verify FRONTEND_URL in backend/.env matches your domain
