BEFORE YOU START:
─────────────────────────────────────────────────────────────────
1. Go to: https://console.firebase.google.com/project/campusmart-7d3d4

2. Enable Firestore:
   • Click "Firestore Database" in left menu
   • Click "Create database"
   • Choose "Production mode"
   • Select location: europe-west1

3. Upgrade to Blaze Plan (for Functions):
   • Click gear icon → Usage and billing
   • Click "Modify plan"
   • Select "Blaze (Pay as you go)"
   • Note: Free tier is generous, likely $0/month

Double-click this file:

  📄 DEPLOY_NOW.bat

This will:
  ✓ Install functions dependencies
  ✓ Build frontend (if needed)
  ✓ Deploy everything to Firebase
