═══════════════════════════════════════════════════════════════
  STEP 1: UPGRADE TO BLAZE PLAN (2 MINUTES)
═══════════════════════════════════════════════════════════════

1. Visit:
   https://console.firebase.google.com/project/campusmart-7d3d4/usage

2. Click "Modify plan"

3. Select "Blaze (Pay as you go)"

4. Add payment method (credit/debit card)

5. Set budget alert at $5 (optional but recommended)

6. Click "Purchase"

DON'T WORRY ABOUT COST:
  • Free tier: 2M function calls/month
  • For small-medium app: $0-5/month
  • You can set spending limits

═══════════════════════════════════════════════════════════════
  STEP 2: ENABLE FIRESTORE (1 MINUTE)
═══════════════════════════════════════════════════════════════

1. Go to:
   https://console.firebase.google.com/project/campusmart-7d3d4/firestore

2. Click "Create database"

3. Choose "Production mode"

4. Select location: europe-west1 (closest to Kenya)

5. Click "Enable"

═══════════════════════════════════════════════════════════════
  STEP 3: DEPLOY TO FIREBASE (5 MINUTES)
═══════════════════════════════════════════════════════════════

Run this command:

  DEPLOY_NOW.bat

This will:
  ✓ Install functions dependencies
  ✓ Deploy frontend to Firebase Hosting
  ✓ Deploy backend to Firebase Functions
  ✓ Deploy Firestore security rules
